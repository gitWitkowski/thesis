## Technologies used
- C++17
- [ROOT v6.30.04](https://root.cern/install/)
- [zlib](https://www.zlib.net/) (zlib1g-dev)
- cmake
 
## How to run
1. Source the appropriate script (if needed)
```bash
source /path/to/root/install_dir/bin/thisroot.sh
```
2.  Make run.sh executable
```bash
chmod +x run.sh
```
3.  Run shell script
```bash
./ruh.sh
```
